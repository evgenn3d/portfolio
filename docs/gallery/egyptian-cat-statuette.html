Hel, Poland

Изящная 3D-модель статуэтки египетской кошки.

Модель создана с помощью фотограмметрии на основе ~100 фотографий, снятых с различных ракурсов. Для получения чистых альбедо текстур применена перекрестная поляризация. Такой подход обеспечивает высокую точность деталей и корректные PBR-текстуры, что делает модель готовой для использования в игровых движках и других 3D-проектах.
